# Problem 1: Lead Finder Is Generating Fake Businesses

Current behavior:

```text
Premium Lopez's clothing store
Gold Martin's clothing store
Vintage Smith's clothing store
```

These are obviously generated placeholders.

A lead finder should NEVER invent businesses.

## What it should do

Real flow:

```text
User enters:
Country
State
City
Business Type

↓
Google Maps Search API
↓
Places API
↓
Fetch real businesses
↓
Store in database
↓
Analyze
```

For example:

```text
Kochi
Roofing Company
```

Should return:

```text
ABC Roofing Kerala
XYZ Roofing Solutions
Kochi Roof Works
```

with

```text
Website
Phone
Rating
Review Count
Google Maps URL
```

---

# Fix

Use:

```text
Google Places API
```

or

```text
Serper Google Maps API
```

or

```text
Apify Google Maps Scraper
```

or

```text
BrightData Maps Scraper
```

Never AI-generate business names.

AI should analyze businesses.

Not create businesses.
